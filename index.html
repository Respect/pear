Respect PEAR Channel Server

Using this Channel

This channel is to be used with the PEAR installer.

Registering the channel: 
pear channel-discover respect.github.com/pear
Listing available packages: 
pear remote-list -c respect
Installing a package: 
pear install respect/package_name
Installing a specific version/stability: 
pear install respect/package_name-1.0.0
pear install respect/package_name-beta
Receiving updates via a feed: 
http://respect.github.com/pear/feed.xml
